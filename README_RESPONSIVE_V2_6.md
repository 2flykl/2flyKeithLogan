# Responsive Platform Update V2.6

- Responsive breakpoints for large desktops, laptops, tablets, phones, landscape phones, and safe-area devices.
- Mobile navigation, hero, carousels, cards, forms, media player, and project sections resize without horizontal clipping.
- Reduced-motion support included.
- Streams now uses translucent thumb controls inspired by modern mobile platform games:
  - left-side circular movement pad
  - right-side circular jump pad
  - controls sit at lower corners rather than across the play field
  - desktop controls become faint and unobtrusive
  - keyboard controls remain Left, Right, and Up
- No control labels are placed across the gameplay view.
