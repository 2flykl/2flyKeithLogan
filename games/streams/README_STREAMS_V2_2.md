# Streams Sprite Engine V2.2

## Gameplay and presentation changes

- Digital-media platforms render at full opacity and retain their original color.
- Removed the labeled caustics/light-pattern source image from the water.
- Media objects now occupy multiple lanes and float downstream at different speeds.
- Some media objects temporarily remain anchored in the stream.
- Large objects are slower and behave like major moving platforms.
- Platform-to-platform separation physics prevents full-width blockades.
- Short-lived clusters are still allowed for challenge and variety.
- Media objects recycle after passing the unseen waterfall below the player.
- Only pennies and blue X balls are collectible.
- Blue attention balls use a white X and no thumb icon.
- Penny collection plays a synthesized cha-ching sound.
- Up Arrow and Space jump.
- Double-press Left or Right and hold the second press for a short momentum dash.
- Opening route remains forgiving while later current speed increases.
