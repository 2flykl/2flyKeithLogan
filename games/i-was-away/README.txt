I WAS AWAY — PLAYABLE BOOMERANG TUTORIAL DEMO

CONTROLS
Left / Right arrows: Aim before throwing; move left/right during return
Hold Down arrow: Pull boomerang back and build power
Release Down arrow: Start forward throwing motion
Press Up arrow: Time the release inside the white meter window
Arrow keys after release: Move into the return area
R: Reset throw
M: Mute/unmute song

HOW TO RUN
Recommended:
1. Install Visual Studio Code.
2. Install the "Live Server" extension.
3. Open this folder in VS Code.
4. Right-click index.html and choose "Open with Live Server."

Alternative with Python installed:
Open a terminal in this folder and run:
python -m http.server 8000
Then visit http://localhost:8000

NOTES
- Internet access is required to load the Three.js library from jsDelivr.
- The included MP3 plays locally from the audio folder.
- Click ENTER THE FIELD to unlock browser audio.
