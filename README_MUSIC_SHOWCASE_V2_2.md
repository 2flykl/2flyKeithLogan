# Music Showcase V2.2

This release redesigns the Music page around a clear selected-album hierarchy.

## Improvements
- Selected album moves into a large left/center showcase position beside the title and description
- Titles and descriptions no longer clip while navigating
- Album controls never cover the selected artwork
- Tracklist and actions appear in a dedicated panel over the faded album-universe side
- Unselected albums are smaller, lower-opacity, and grayscale
- Previous/next arrows remain highly visible
- Clicking any small cover promotes it to the selected showcase
- All visible “Support” wording is replaced with “Help Me Create”
- Existing navigation, audio crossfades, experiences, and player remain intact

## Install
Copy all files and folders into the root of the local `2flyKeithLogan` repository and replace existing files.
