# Hero Carousel 2.0 + Interactive Audio Engine

## Install
Copy every file and folder from this package into the root of your local `2flyKeithLogan` repository. Replace existing files, but do not delete `.git`.

Commit message:
`Add dramatic carousel and interactive music soundscape`

Push to GitHub, wait for Pages deployment, then hard-refresh with Ctrl+F5.

## New behavior
- Selected hero cover is substantially larger and uses pointer-driven parallax.
- Controls sit directly beneath the selected project information.
- Each hero project has a distinct particle universe.
- Music cards enlarge, tilt, reveal controls, and alter the page theme on hover/focus.
- Soundscape Mode crossfades short song previews after the visitor activates it.
- Preview audio is opt-in to comply with browser autoplay rules.
